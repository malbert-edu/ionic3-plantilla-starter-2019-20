Bienvenido al proyecto Ionic/Angular!

Este proyecto es la plantilla base para comenzar a desarrollar una aplicación web basada en Angular e Ionic.

  • Utilizaremos Angular para desarrollar toda la lógica de la aplicación. Angular es un potente framework de desarrollo en JavaScript que simplifica mucho la programación de aplicaciones web.
  • Ionic es un framework que puede utilizarse junto con Angular que nos ofrece muchas características y funcionalidades muy fáciles de utilizar y que permiten, entre otras cosas, una mejor navegación por la aplicación y un diseño mucho más nativo e intuitivo.
